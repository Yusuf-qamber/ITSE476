Installation Guide

Follow these steps to set up and contribute to the open-source Bite Finder project.

Here‚Äôs how to set up Bite Finder on your computer:

  1. Fork the project from our GitHub repository.
  2. Clone it to your computer:
    git clone https://github.com/Yusuf-qamber/Bite-Finder.git
  3. Open the folder and go to either the frontend or backend:
    cd Bite-Finder/bite-finder-frontend

    or

    cd Bite-Finder/bite-finder-backend
  4. Install the needed packages:
    npm i
  5. Set up your .env file and add it to .gitignore
  6. Create a new branch:
    git branch "yourBranchName"
  7. Make your updates or fixes.
  8. Submit a pull request for review.
